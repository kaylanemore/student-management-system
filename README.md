# Student Management System

Simple Java console application for managing students and their grades.

## Technologies

* Java
* IntelliJ IDEA
* Git

## Features

* Add students
* List students
* Store student name and grade
* Basic service layer for managing data

## Project Structure

src
├─ Main.java
├─ Student.java
└─ StudentService.java

## How to Run

1. Clone the repository

git clone https://github.com/SEU-USUARIO/student-management-system.git

2. Open the project in IntelliJ IDEA

3. Run the Main.java file

## Purpose

This project was created for learning Java, object-oriented programming, and Git version control.
